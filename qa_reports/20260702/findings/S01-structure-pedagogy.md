## 八、结构亮点（正反馈）

1. **导读→小结闭环高度一致**。10 章全部兑现导读承诺，导读立的悬念在小结中得到回答。这是本书结构最扎实的地方。

2. **章末承上启下链完整**。每章末尾都有对下一章的明确预告，且预告内容与下一章实际内容匹配。读者能清晰感知"现在在哪里、接下来去哪里"。

3. **Ch05 "三层统一视角"映射**。将 Redis(networking.c + server.c + db.c)、MySQL(sql/ + storage/)、Kafka(network + api + log) 各自不同的源码模块映射到"交互层/逻辑层/存储层"统一视角，是全书最好的抽象映射之一，让三款差异巨大的软件有了可比的共同语言。
