## 核心约束（详见 AGENTS.md §6）

**总调度只做调度**：任何实质性分析/审查必须派子 agent，不得 inline 自己读自己判。

**只读分析（不改文件）**：≥2 个不同视角 agent 独立判断，结论一致才采信。

**改动文件**：改前多 agent 分析（≥2 同意才动）→ 改时单人执行（用 Edit，精准替换）→ 改后多 agent 验证。

**其他**：语义类不得 grep 自证 / 改字词不改意思 / 绝对保留作者第一人称案例与判断锋芒。
